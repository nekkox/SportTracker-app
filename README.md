# SportTracker-app

Sport Fitness tracker where you can store and track your excercising results.  
Data is stored in Supabase.
You can login with OTP (One time password) - your email. Or use test account.

---

## Tchnologies
- Vue 3
- Vue-Router
- Vuetifyjs
- Vite
- Pinia
- Supabase
- vue-chartjs
